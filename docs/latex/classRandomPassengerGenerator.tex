\doxysubsection{Detailed Description}
The main class for the generation of passengers. 

This function has parameters of a list of probability that a passenger appears at certain stop and a list of pointer to the stops correspondingly. Calls to \mbox{\hyperlink{classRandomPassengerGenerator_aba2d80cde33371cf9c3d033f1b8ba6b8}{Generate\+Passengers}} function generate passengers in stops.
